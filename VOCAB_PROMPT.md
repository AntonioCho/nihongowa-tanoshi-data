#목표: 입력된 일본어 단어 목록을 JSON 배열로 변환한다.
#규칙:
1. 입력된 단어 순서를 유지한다.
2. 단어는 사전에 있는 것으로 한정한다. 해당 단어를 포함하는 예문을 만들 수 없는 표현인 경우에는 제외하고 빈 객체 {} 대신 해당 위치에 null을 반환한다.
3. 입력의 의미(meaningKo)는 무시한다. 단어의 가장 많이 사용되는 의미 하나를 선택하여 meaning, 예문에 활용한다.
4. 후리가나 형식: ["표시텍스트", "후리가나"] 쌍의 배열. 히라가나/가타카나/기호는 ["문자", ""] 형태.
5. 반드시 입력 단어 수와 동일한 수의 객체를 JSON 배열로 반환한다. 다른 형식(중첩 객체, 래퍼 키 등)은 사용하지 않는다.

#입력 형식:
단어|품사(한글)|의미(한글)
(한 줄에 한 단어, 여러 줄)

#출력 형식: JSON 배열 (배열 외 다른 구조 금지)
[
  {
    "kanji": "あげる",
    "meaning": "to give",
    "meaningKo": "주다.",
    "furigana": [["あげる", ""]],
    "subject": "기초 동사",
    "example": {
      "sentence": "友達に本をあげる。",
      "meaning": "I give a book to my friend.",
      "meaningKo": "친구에게 책을 준다.",
      "furigana": [["友達", "ともだち"], ["に", ""], ["本", "ほん"], ["を", ""], ["あげる", ""], ["。", ""]]
    }
  },
  {
    "kanji": "次の単語",
    ...
  }
]
